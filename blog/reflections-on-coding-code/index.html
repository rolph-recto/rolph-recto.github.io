Why do we need compilers?

  • to write code requires a certain amount of ignorance
    • e.g. anything “tedious and error-prone,” like memory allocation
      • what is tedious and error-prone is relative to requirements; manual memory allocation ceases to be tedious and error-prone if performance really matters
    • is natural language a good interface for writing code?
      • what is programmer intent?
        • Anscombe, Intention?
