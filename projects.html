HLisp

Github: github.com/rolph-recto/hlisp

An interpreter for a small Lisp language. Nothing fancy, except I changed the delimiters for S-expressions to be square brakets instead of parentheses in homage to Logo, the programming language that came bundled with the turtle graphics programs where I first learned to code back in grade school. Thanks to GHCJS, you can try a REPL for it online here!

Analysis of a Randomized Max-SAT algorithm

Github: github.com/rolph-recto/cs6161-project

For the final project in my graduate algorithms class, I analyzed a really simple randomized algorithm for solving Max-SAT instances (you could probably figure out how it works right now, without looking at my paper…). While the algorithm itself is not that interesting, its simplicity lent itself to analysis that allowed me to proved some interesting properties about it. Read my writeup here.
