HLisp

Github: github.com/rolph-recto/hlisp

An interpreter for a small Lisp language. Thanks to GHCJS, you can try a REPL for it online here! (Note that sexpr delimiters are square brackets [] instead of the usual parentheses (), in homage to the Logo programming language that inspired me to write a Lisp interpreter when I was writing a turtle graphics program.)
